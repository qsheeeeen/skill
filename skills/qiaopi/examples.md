# 侨批范例

以下皆为真实侨批风格的往来书信，出自木生与淑柔夫妇。写批时请模仿其语气、格式与时代感。

---

## 范例一：简短报平安 + 寄银

```
收信人：淑柔我妻

正文：付港币五十元，随寄布料十尺。我在暹罗非常好，免担忧。

寄信人：吾夫木生
```

> 最简短的侨批。报平安、报金额、报物品，三句话。暹罗即泰国旧称。

---

## 范例二：妻子回信，叙家常

```
收信人：吾夫木生

正文：信件及家用已悉数收到，家中诸事皆顺，庄稼长势尤好，番薯肥美壮实。昨日将其烤作晚餐，孩子们十分喜爱。大弟最馋嘴，半夜摸黑，又偷啃番薯，声响吵醒我们。大妹气得直跺脚，细弟更是严厉，强开其口，务必查明罪证，终于水落石出。不过多时，三人又嬉闹起来，待鸡鸣声响，方才睡去。望你见此，定能感同身受，因这三个天真可爱的孩子倍感幸福。暹罗炎热，多加保重身体。

寄信人：妻，淑柔
```

> 妻子回批。不提辛苦，只说孩子琐事让丈夫如临其境。结尾嘱托保重——侨批的典型收束。

---

## 范例三：叙事 + 抒怀

```
收信人：（称呼缺失，推测为「吾妻淑柔」）

正文：与妻一别，八载有余，日思夜想，归期遥遥，唯化思念作拼搏，凭勤俭来立业。今把三轮换货船，终得扬帆起航。江海有岸，团圆可盼。

寄信人：夫，木生
```

> 讲述奋斗进展，让妻子看到团圆的希望。叙事简练，有分量。

---

## 范例四：长篇书信，见闻 + 抒情（推荐模板）

```
收信人：吾妻淑柔，展信安康

正文：随信寄两百元，我一切无恙，生意昌顺。行船入夜，恰江上升明月，圆如玉坠，仿若身在故乡，似与你并肩共赏。江海万里，心中念你，便不觉遥远。湄南河畔木棉花盛开，像极了家乡的春天，压了一朵在信中，望你也能闻到花香。近来握笔练字，学会了你的名，虽然潦草，努力数日定会成功。纸短情长，伏惟珍重。

寄信人：夫，木生
```

> 最完整优美的范例。寄银→报平安→见闻（江月、木棉）→心事（练写妻名）→收束。200 字左右，结构完美。

---

## 范例五：妻子回信，长篇

```
收信人：吾夫木生，展信佳

正文：一百元已收到，七月初七，大妹出花园，已亭亭玉立。大弟与小弟，亦个头出挑，健朗聪慧，见子女茁壮成长，欣慰非常，这是你我共修之骄傲。七夕当夜，你衣锦归来，仍是少年模样。梦醒行至寨门前，闻溪水潺潺，方觉夜深，念你安康，好梦，即已知足。

寄信人：妻，淑柔
```

> 收银确认→子女近况→梦境寄情→收束祝福。梦中相逢的笔法含蓄深情。

## 侨批短句摘录

以下零散句子可化用于正文：

- 「暹罗在这头，唐山在那头，你在我心里头」
- 「冬至将至，虽你未能归，冬至丸亦留你一份」
- 「打了新棉被，眠床烧烧，不畏天寒，你免挂念」
- 「暹罗虽远，心有所寄，身若比邻，切要平安，即为团圆」
- 「我心只有一个，一心不能二用，今生可与你结伴，我已万分满足」
- 「家里大小平安，但两个月没收到你的来信，望速报安康，免我心忧」
- 「听到要去汕头市区照相，三个小孩高兴得睡不着，天未亮就出门」

---

## 风格总结

1. **称呼**：亲密用「吾妻/吾夫 + 名字」，敬长辈用「父母亲大人膝下」
2. **正文**：先报平安 + 寄银数额 → 叙事一二 → 嘱托收束
3. **语气**：不煽情，不夸张。用日常小事传情（番薯、棉被、冬至丸）
4. **收束**：纸短情长、伏惟珍重、免挂念、切记保重
5. **落款**：简洁，「夫，木生」「妻，淑柔」「儿 ×× 叩上」
6. **时代词**：暹罗（泰国）、叻币/港币、番薯、出花园（潮汕成人礼）、眠床（床）、唐山（中国）
